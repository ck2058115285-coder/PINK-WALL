# PINK WALL — 极简相册与瀑布流摄影主题

**PINK WALL** 是一款专为 Typecho 打造的高颜值、现代且极简的响应式瀑布流摄影相册主题。主题兼顾文字博客与大图画廊的展现，内置多款沉浸式交互动效，提供极致的跨设备阅读体验。

---

## 🌟 主题特色

1. **响应式自适应瀑布流**：使用极低能耗的 CSS Column 瀑布流布局，在 PC、平板、手机等不同端完美对齐、防截断，性能拔群。
2. **沉浸式日夜双色模式**：支持一键切换白天与黑夜模式，内置平滑的过渡动画，日夜模式状态自动锁定至本地缓存。
3. **首屏全景大图幻灯片**：
   - **16:9 黄金限高比例**与边栏无缝重合的融屏设计，顶栏高透毛玻璃背景，极具质感。
   - **API 实时随机打乱加载**：每次刷新自动从内置的 20 张世界级经典作品或您的自定义列表中切片抽取 6 张随机展现，防浏览器强缓存，降低网络带宽开销。
   - **完全可控**：支持在外观设置中一键开启/关闭幻灯片，并支持可视化自定义轮播图片及文字描述。
4. **内置原生灯箱大图浏览器 (Lightbox)**：
   - 零依赖、极速响应。图集内页与文章卡片多张大图支持弹窗放大，支持鼠标左右切换及键盘方向键（`←`/`→`/`ESC`）交互。
5. **智能动态缩略图缓存生成**：
   - 内置基于 PHP GD 库的动态缩略图缓存服务。当加载瀑布流网格时，系统自动生成 600px 宽度的小尺寸缓存图（保存在 `usr/uploads/cache/`），显著减小流量与首屏加载时长；而激活 Lightbox 灯箱时则继续加载无损高清大图。
6. **骨骼屏微光与旋转双重加载动画**：
   - 完美解决体积较大的图片在网络加载期间白屏或灰色占位死寂的问题，图片加载成功后平滑淡入。
7. **网站稳定运行计时器**：
   - 支持在后台外观设置中自定义建站时间（如 `2019-03-30`），页脚自动秒级动态渲染运行天数。

## 📸 原生相册瀑布流页面使用说明

主题中全新开发了**原生相册瀑布流模板 (`page-photo.php`)**。

### 如何启用您的相册：
1. 登录 Typecho 后台，点击 **“控制台”** ➔ **“新建页面”**。
2. 填写页面标题（例如 `Photo` 或 `相册`），在右侧的 **“自定义模板”** 选项中，下拉选择 **“相册瀑布流 (Photo Gallery)”**。
3. 在页面下方点击 **“上传附件”** 并批量上传您想要在相册中展示的所有照片。
4. 点击发布页面。
5. 之后模板会自动抓取该页面关联的所有照片附件，并在前台自动以响应式瀑布流整齐铺满展现。

### 后续如何添加新照片：
1. 后续您需要添加新照片时，**无需**修改任何代码。
2. 直接进入后台 **“管理页面”**，编辑您的这个相册页面，继续在下方 **“上传附件”** 上传新照片即可。
3. 您可以自由拖动或在附件管理中调整照片的顺序，模板会自动为您按最新的顺序动态渲染。
4. 如需为照片添加描述，可直接在后台附件列表中编辑该图片的“描述”，它将自动作为悬停注解和灯箱字幕展现在前台。

## 🔗 优雅的自定义友情链接独立页面使用说明

主题新增了**友情链接页面模板 (`page-links.php`)**。

### 如何启用友情链接页面：
1. 登录 Typecho 后台，点击 **“控制台”** ➔ **“新建页面”**。
2. 填写页面标题（例如 `Links` 或 `友链`），在右侧的 **“自定义模板”** 选项中，下拉选择 **“友情链接 (Friend Links)”**。
3. 在页面编辑框内，您可以撰写您的申请友链格式、说明、寄语等。
4. 在说明文字下方，输入一条普通的**分割线**（Markdown 中输入三个减号 `---`，富文本编辑器中点击分割线图标即可）。
5. 在分割线下方，按以下格式每行输入一条友链：
   ```text
   博客名称|博客链接|头像URL|一句话介绍
   ```
   * *例如*：`Google|https://www.google.com|https://www.google.com/logo.jpg|GOOGLE`
   * *注：如果对方未提供头像或介绍，可以留空（例如：`Google|https://google.com||`）。*
6. 点击发布即可。

## 🛠️ 安装与启用方法

1. 将本主题压缩包解压或上传至 Typecho 安装目录 of `/usr/themes/` 路径下，并确保文件夹名称为 `pink-wall`。
2. 登录 Typecho 后台，进入 **“控制台”** ➔ **“外观”** ➔ 找到 **“PINK WALL”** 点击 **“启用”**。
3. 启用后点击 **“设置外观”** 可以进行如下配置：
   - **主页大图幻灯片开关**：开启或关闭首屏巨幕。
   - **自定义大图幻灯片配置**：按行填写自定义轮播图（格式：`图片链接|分类|标题|作者|出处链接`），留空则自动随机展示内置的大师之作。
   - **建站起始时间**：填入您的建站时间（如 `2019-03-30`），用于自动展示建站时长。
   - **站点 LOGO 与大图横幅文字**：自定义前台首屏展示的大标题与介绍。
4. **性能建议**：建议在服务器的 PHP 环境中开启 **GD 图像库扩展**（即在 `php.ini` 中取消 `;extension=gd` 的注释并重启 PHP 服务），以便主题能够自动为您在瀑布流中生成并载入极速小体积缩略图。
